# wisspar

WISSPAR: R project scaffolded for PneumococcalCapsules/WISSPAR.

## Structure
- R/: R scripts and functions

## Setup
Run the following from the project root to initialize and push (requires GitHub auth):

```powershell
git init
git branch -M main
git remote add origin https://github.com/PneumococcalCapsules/WISSPAR.git
git add .
git commit -m "Initial commit: scaffold wisspar project"
git push -u origin main
```
